This folder contains code to explore various aspects of models, simulation studies, etc. 

* `01_models`: where we try to understand how to use new models. We also do this for models that may not be used for our simulations, but provide viable alternatives. This folder contains the data generators from the simulation studies used in the research report. 
* `02_data_generation`: here we try to design the scripts that build (simulate) the data that we use for our simulation studies.
