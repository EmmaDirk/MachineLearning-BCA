In this folder are helper functions. Some are used in simulation engines. Others are just used to understand things about this thesis. The following helpers are currently available:

* `helper_functions_jeroen`: a collection of helper functions used to simulate panel data. 
* `simulation_function_jeroen`: the main simulation function used in the simulation studies.
* `visualing_thin_surface`: a script to visualize negative sampling in high-dimensional space.
* `sampling_betas`: a function to calculate the confounder strength from a beta matrix. A function to compute the ammount of non-linearity in a beta matrix. A function to sample beta matrixes with a given confounder strength and non-linearity (this function has been updated since).
