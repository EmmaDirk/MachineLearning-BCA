This repository contains the Thesis project of Mano. It has the following folders:

* `00_notes`: contains notes and references used during the thesis development.
* `01_scripts`: contains the scripts used in the development of the simulation studies.
  * `00_legacy`: contains deprecated scripts that may be useful in the future.
  * `01_helpers`: contains helper functions used in the simulation engines, or that just serve to visualise complicated things.
  * `02_development`: contains the simulation engines under development for full simulation studies.
* `02_data`: contains the data generated during the simulation studies.
* `03_output`: contains the output pictures generated during the simulation studies.
* `04_presentations`: contains intermediate results discussed on weekly basis.
* `05_writing`: contains the writing of the thesis and intermediate products, feedback and literature. 
* `o6_planning`: contains files outlining what the thesis project entails, due dates, etc.
